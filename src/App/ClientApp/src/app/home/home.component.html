Para migrar la base de datos por favor realizar los siguientes pasos

  • Ve a la carpeta de Nexos.src.App
  • Ingresa los siguientes comandos
  • dotnet ef migrations add Inicial --context NexosContext -p ..\Infraestructure -s App.csproj -o Data.Migrations -v
  • Y acontinuacion ingresa el comando dotnet ef database update -c NexosContext -p ..\Infraestructure -s App.csprojs -v
